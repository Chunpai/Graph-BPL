BWSN dataset contains 3 folders:

1. rawData folder:
	1. ./rawData/edges.txt : contains the edges (links between junctions) of the BWSN graph
	2. ./rawData/vertices.txt : contains vertices (junctions) of the BWSN graph.

2. testGraphs: contains all of the testing graphs (graph with noise 2%,4%,6%,8%,10% in 3 hours)
3. trainingGraphs: contains all of the training graphs (graph without noise in 3 hours)
